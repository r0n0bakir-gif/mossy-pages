# 🌿 Mossy Pages

A cozy pixel art journal app — single HTML file, no install needed.

## Features

- **Pixel art aesthetic** — Press Start 2P font, retro animations, CRT scanlines
- **Folders** — organise entries into Daily Life, Self Improvement, Dreams, Gratitude, Creative (or create your own)
- **Remembers everything** — entries saved automatically via localStorage
- **Fully portable** — click Export to download a self-contained `.html` snapshot of your journal with all data baked in

## Usage

Just open `cozy-journal.html` in any browser. No server, no install, no dependencies.

To export a portable copy with all your entries: click the **Export** button in the top-right corner.
